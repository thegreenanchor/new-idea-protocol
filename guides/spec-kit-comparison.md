# Spec Kit Comparison

Spec Kit is an implementation workflow for a project repo.

New Idea Protocol is the intake workflow before a project repo exists.

Use `/new_idea` when:

- The idea is still vague.
- The user flow is not clear.
- MVP boundaries are missing.
- The agent would otherwise guess.

Use Spec Kit or another implementation workflow when:

- A project repo exists.
- The feature can be specified as build tasks.
- Architecture and constraints are ready for implementation.

New Idea Protocol can hand off into Spec Kit, Seven-File Context, Claude Code, Codex, Cursor, or a human engineering process.
