# Phase 1: Intake

Ask for or infer:

1. Raw idea in the user's own words.
2. Domain or brand.
3. Topic/title.
4. Whether to create a Wizard-of-Oz prototype.

Prompt:

```text
I will run this through New Idea Protocol before any code is written.

First, I need the raw idea, domain, topic/title, and whether you want a Wizard-of-Oz prototype: a quick clickable simulation to test the workflow before building the real system.
```
