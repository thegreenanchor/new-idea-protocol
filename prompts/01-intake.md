# Phase 1: Intake

Ask for or infer:

1. Raw idea in the user's own words.
2. Domain or brand.
3. Topic/title.
4. Which visual artifacts to create: visual spec, Wizard-of-Oz prototype, both, or neither. Ask once.

Prompt:

```text
I will run this through New Idea Protocol before any code is written.

First, I need the raw idea, domain, topic/title, and which visual artifacts you want: a visual spec (styled one-page HTML overview of the idea), a Wizard-of-Oz prototype (quick clickable simulation to test the workflow), both, or neither.
```
