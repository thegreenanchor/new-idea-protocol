# Personal Command Center Spec

## MVP promise

Turn one messy transcript into a clear project brief, assumptions list, decisions list, and top three next actions as Markdown.

## Primary user and success moment

A solo operator pastes a transcript and gets a usable project brief in under two minutes.

## In scope

- Paste transcript.
- Generate brief.
- Extract assumptions and decisions.
- Export Markdown.

## Non-goals

- No mobile app.
- No team accounts.
- No calendar sync.
- No automatic task creation.

## Data and storage

Local files store transcript, generated artifacts, and export history.

## Acceptance criteria

- [ ] User can paste a transcript.
- [ ] User receives four Markdown artifacts.
- [ ] Output includes assumptions and decisions.
- [ ] Exported Markdown can be copied into a notes app.
