# Repo Context Handoff

Use this when a clarified idea is ready to become a project repo.

## Required root files

- `AGENTS.md` or `CLAUDE.md`
- `context/project-overview.md`
- `context/architecture.md`
- `context/code-standards.md`
- `context/ai-workflow-rules.md`
- `context/ui-context.md` or `context/interface-context.md`
- `context/integrations.md`
- `context/progress-tracker.md`

## Rule

Do not write feature code until the context files exist and the MVP boundaries are confirmed.
