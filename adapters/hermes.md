# Hermes Adapter

In a Hermes setup, install this as a skill or keep the prompt files nearby.

Private Obsidian-style routing can save specs to:

```text
C:\Users\<you>\path\to\vault\Sources\_inbox\<Topic>.md
C:\Users\<you>\path\to\vault\Sources\_inbox\<Topic>-prototype.html
```

Keep the canonical SOP in the user's private knowledge base if they have one. Root public docs should stay generic.

Hermes-specific rule: after a non-trivial private wiki edit, update the relevant wiki log before final delivery.
