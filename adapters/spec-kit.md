# Spec Kit Adapter

Use New Idea Protocol before Spec Kit when the idea is still vague.

Suggested handoff:

1. Run `/new_idea` until `SPEC.md`, `ASSUMPTIONS.md`, `DECISIONS.md`, and `HANDOFF.md` exist.
2. Create the project repo.
3. Start Spec Kit from the clarified spec rather than from the original rough idea.
4. Keep non-goals and assumptions visible during implementation.
